La boussole

Partager vos traces GPS

Que ce soit pour aller marcher en famille, randonner en montagne ou courir dans les bois c'est toujours plus confortable de suivre notre gps. Le but de la-boussole est de vous proposer une manière innovante de choisir la trace GPS qu'il vous faut.

    Fonctionnalitées prévues :
  • Visualiser la trace GPS sur une carte Top25.
  • Visualiser le dénivelé sur un graphique intéractif.
  • Rechercher les traces près de chez moi.
  • Proposer une variante.
  • ...
 
 

Vue technique

  • la-boussole est basée sur le nouveau language Dart :
    • platforme permettant d'utiliser le même language coté serveur et coté navigateur.
    • langage moderne exploitant les retours d'expériences des "vieux" languages.
  • la-boussole est construite, deployée et hebergée grâce des services performants disponibles sur le web :
    • GitHub.org pour la gestion des sources.
    • Drone.io pour l'intégration continue et le deployement continu.
    • Heroku.com pour l'hébergement de l'application.
  • l'architecture de la-boussole est basée sur :
    • MongoDB pour le stockage de données hétérogènes.
    • Platforme Dart avec les librairies : objectory, rikulo...
    • GeoPortail API pour naviguer sur les fameuses cartes Top25.
    • Javascript compilé depuis le code Dart.
    • WebComponent sur Polymer: c'est l'avenir du Web.
  • Enfin the backlog est disponible sur Trello.
